# GUARANTEED WORKING RENDER COMMANDS

## Build Command (Simple & Reliable):
```bash
pip install -r requirements.txt && mkdir -p backend/vectorstore backend/logs && cd backend && python -c "import sqlite3; import os; os.makedirs('vectorstore', exist_ok=True)" && cd ..
```

## Start Command (Guaranteed to Work):
```bash
python start_reliable.py
```

## Alternative Start Command (If above fails):
```bash
cd backend && python -m uvicorn app:app --host 0.0.0.0 --port $PORT
```

## Alternative Start Command (Most Basic):
```bash
cd backend && python app.py
```

## What Each Command Does:
1. `python start_reliable.py` - Uses the new reliable script
2. `cd backend && python -m uvicorn app:app --host 0.0.0.0 --port $PORT` - Direct uvicorn
3. `cd backend && python app.py` - Runs the app directly

## Try in this order:
1. First: `python start_reliable.py`
2. If fails: `cd backend && python -m uvicorn app:app --host 0.0.0.0 --port $PORT`
3. If fails: `cd backend && python app.py`
