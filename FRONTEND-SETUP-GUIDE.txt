# HOW TO GET YOUR FRONTEND WORKING

## The Problem:
Your backend is working, but the frontend is not starting because it needs to be built first.

## Solution: Build Frontend Locally

### Step 1: Build Frontend on Your Computer

#### Windows:
```bash
build-frontend-simple.bat
```

#### Or manually:
```bash
cd frontend
npm install --legacy-peer-deps
npm run build
```

#### Linux/Mac:
```bash
cd frontend
npm install --legacy-peer-deps
npm run build
```

### Step 2: Commit the Built Frontend
```bash
git add frontend/out/
git commit -m "Add built frontend"
git push
```

### Step 3: Update Render Commands

#### Build Command:
```bash
pip install -r requirements.txt && mkdir -p backend/vectorstore backend/logs && cd backend && python -c "import sqlite3; import os; os.makedirs('vectorstore', exist_ok=True)" && cd ..
```

#### Start Command:
```bash
python start_with_frontend_fixed.py
```

## What You'll Get:
- ✅ Beautiful Persian chatbot interface at root URL
- ✅ Backend API at /api
- ✅ Both working together

## Alternative: Quick Test
If you can't build frontend right now, test your API:
