Zimmer Chatbot Widget Test Page

Test Instructions

This page is for testing the Zimmer Chatbot widget. You should see a floating chat button in the bottom-right corner.

Click the button to open the chat window and test the chatbot functionality.

Embedded Widget Snippet:

<!-- Zimmer Chatbot Widget -->
<div id="zimmer-chatbot-root"></div>
<script>
  (function () {
    window.ZIMMER_CHATBOT_CONFIG = {
      apiBaseUrl: "https://chatbot.zimmerai.com",
      siteId: window.location.host
    };

    var script = document.createElement("script");
    script.src = "https://chatbot.zimmerai.com/static/chatbot-widget.js";
    script.async = true;
    document.head.appendChild(script);
  })();
</script>
